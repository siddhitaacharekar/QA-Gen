3 ways to run tests — pick your comfort level
🎬 Walkthrough tab — zero setup beginner
Click "Run all tests" — QAGen simulates each step visually, raises bugs on failures. No installation needed.
⚡ Local proxy — real browser intermediate
Run node proxy.js once. Playwright opens a real browser and tests your actual app. Results come back to QAGen.
🔄 CI/CD — runs on every PR automated
In BDD tab → Step 3 → Download GitHub Actions workflow. Merge it once — tests auto-run on every pull request forever.
